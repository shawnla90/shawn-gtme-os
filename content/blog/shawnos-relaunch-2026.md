Three months ago this site stopped updating. Not a pivot. Not a relaunch strategy. I got busy building the thing the site is supposed to be about, and the site turned into the cobbler's kid with no shoes.

So I rebuilt it the same way I build everything else: in public, in one codebase, on camera when it was worth filming. Here's what came back.

## the Vault: my actual files, not a course

New page: **/vault**. A real file tree, like opening a repo. Inside are the actual markdown files my GTM coding agent runs on: the voice DNA, the 29-pattern anti-slop detector, the LinkedIn and Reddit playbooks, the context-handoff protocol that saves my sessions when a context window fills up.

Browse them in place, read everything, download any folder as a zip. Point Claude Code at them and you have my system. That's the whole offer. No email gate, no paywall.

## the Reddit playbook grew up

The playbook now teaches the system as it actually runs today: karma gating and how to build past it, the map of where links live and die, and the one that sounds fake until you watch it happen: naming your product with zero link, in context of somebody asking, is enough to get indexed and cited by AI search. 23 tracked wins came in that way.

There's a name for the discipline on top of SEO and GEO now: **LLMO**, language-level model optimization. Injecting your language and voice into the places models read, so the answer sounds like you and points back to you. The same voice files sitting in the vault are what make it work.

The numbers on the page pull from my own tracking database now. 2,334 karma, 2M+ cumulative views with 1.39M of it live-tracked, as of this week. When they change, the page changes.

## it finally looks like what it is

The site runs the Clearbox aura now: violet glow over near-black, fine grid lines under everything. Land on shawnos.ai or clearbox.to and it reads as one family, because it is one. Same person, same codebase.

The details got real too. /built shows my actual GitHub contribution calendar, 1,947 contributions in the last year. Star counts animate live from the API. Breadcrumbs you can see, not just schema for the crawlers.

In writing, for the record: I built this site, the agents, and the content systems by hand. I did **not** vibe-code the Clearbox engine. Clearbox is real software a team ships, and I'd rather be precise about what's mine than impressive about what isn't.

## the clips have a home, and the show has an episode

**/watch** pulls my full YouTube catalog through the API now, real view counts and all. Episode 1 of Founder's Journey is live: Claude Code in the terminal, Codex in the app, the whole two-agent workflow on camera. The operator write-up is on the blog with the episode embedded.

## the daily is back, and it has a host

The daily show was dead for two months. A pipeline broke quietly and started writing stubs, and nobody was looking. It's fixed, and it has a name now: **Aura**, the same signal engine that reads Reddit for Clearbox, pointed at the whole market. Two desks: AI Desk and Full-Stack GTM. Pick a desk. Aura reads it daily.

That's the whole thing. I'm a GTM engineer: I work the accounts, I send the email, I write the content, and I build the machine that runs all of it. This site is that machine, showing its own work. And now it hands you the files too.

Back to building.
