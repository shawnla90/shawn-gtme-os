## Profile Creation Checklist

When creating each profile:
1. Use exact brand name: "GTMe OS" (or "ShawnOS" for personal brand profiles)
2. Use consistent URL format (https://thegtmos.ai)
3. Add website link immediately
4. Link the new profile FROM your website footer/JSON-LD
5. Record URL in this tracker
6. Wait 1-2 weeks, then check Google indexing (`"profile-url"` in quotes)
7. If not indexed after 2 weeks, link the profile from another indexed domain
