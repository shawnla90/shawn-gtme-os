# Strategy Rules

## Budget
- Starting capital: **$100**
- This is learning money. Assume it can go to zero.

## Position Rules
- Max single position: 30% of portfolio ($30 at start)
- Never go all-in on one asset
- Always keep a cash reserve (minimum 20%)

## Entry Rules
- Only act on `consider_buy` signals (never on `watch` alone)
- Verify signal reasoning makes sense to YOU before acting
- Wait for at least 2 consecutive signal runs showing the same direction
- Dollar-cost average in — don't lump-sum on a single signal

## Exit Rules
- Take profit at +20% on any position
- Cut loss at -15% on any position
- If `consider_sell` signal fires twice consecutively, evaluate exit

## Meta Rules
- Review signals daily but don't feel obligated to act
- Journal every trade decision (including decisions NOT to trade)
- The system is for learning, not for getting rich
- If the system says hold, hold. Boredom is not a signal.
