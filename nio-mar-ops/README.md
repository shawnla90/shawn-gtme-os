# Nio Marketing Operations

NioBot V3 marketing strategy, campaigns, content, and channel ops.

## Structure

```
nio-mar-ops/
├── strategy/
│   ├── MASTER-PLAN.md          ← Full marketing strategy
│   ├── sprite-evolution-map.md ← Avatar IP versioning plan
│   └── positioning.md          ← Nio brand positioning
├── campaigns/
│   ├── v3-launch.md            ← V3 product launch campaign
│   ├── tamagotchi-teaser.md    ← Evolution system teaser campaign
│   └── niobot-ai-launch.md    ← Domain launch campaign
├── content-bank/
│   ├── launch/                 ← Launch day content (ready to ship)
│   ├── social/                 ← Evergreen social posts
│   ├── threads/                ← Long-form X threads
│   └── scripts/                ← Video/TikTok scripts
├── assets/
│   ├── sprite-audit/           ← Current sprite inventory + gaps
│   └── brand-kit/              ← Colors, fonts, usage guidelines
├── channels/
│   ├── x-twitter/              ← X bot strategy + content calendar
│   ├── discord/                ← Community playbook
│   ├── rss/                    ← RSS feed strategy
│   └── niobot-ai/              ← Domain content plan
├── automation/                 ← Bot configs, cron content jobs
└── data/                       ← Metrics templates, tracking
```

## Build Order

1. Product phase: Ship V3 (messages, chimes, tamagotchi)
2. Pre-launch: Sprite audit, brand kit, content bank
3. Launch: niobot.ai, X campaign, Discord community
4. Sustain: Automated content, RSS, community growth
