# Firecrawl - Website Scraping

> Deep page analysis for enrichment and scoring.

## Uses

- `find_similar.py` scoring: scrape prospect websites for content quality assessment
- Brand color extraction: CSS/meta theme-color for landing page accuracy
- Gap analysis: site audit data for Touch 2 content

## Planned Uses

- Technographics: detect tools via script tags, meta tags
- Content audit: assess blog frequency, content quality, SEO basics
- Competitive analysis: scrape competitor customer pages
