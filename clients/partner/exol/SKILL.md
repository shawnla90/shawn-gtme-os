# Exol (GreenBox Logistics) — GTM Operations Skill Tree

## Company Overview

**GreenBox Logistics** (internal project name: Exol) is a "Warehouse-as-a-Service" provider that decouples high-tech automation from capital expenditure.

**Core offering**:
- Nationwide network of highly automated fulfillment centers (backed by **$7.5B Symbotic partnership**)
- Brands access enterprise-grade robotics and AI without building their own facilities
- **Consumption-based pricing** — pay-as-you-go (AWS model), not rigid long-term leases

**Products**:
1. **Automated Fulfillment**: Pallet, case, and unit-level picking using Symbotic robots
2. **GreenBox Fulfillment AI**: Proprietary software layer for real-time visibility and predictive compliance
3. **Consumption-Based Pricing**: Variable cost model that eliminates CapEx barriers

**Value prop**: "Capital-Free Automation" — enterprise automation without the $100M+ build cost.
